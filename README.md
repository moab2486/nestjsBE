## Description

A restful API built with nestJs framework

## Installation

```bash
$ npm install
$ cp .env.example .env
```
## Migration

copy and run the media.sql file to migrate db

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
